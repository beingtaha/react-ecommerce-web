# React E-Commerce Web App

A mini e-commerce web application built using React JS.

## Features
- Product listing using DummyJSON API
- Product detail page
- Add to cart functionality
- Cart page with selected products
- React Router based navigation

## Tech Stack
- React JS
- JavaScript (ES6)
- HTML
- CSS
- DummyJSON API

## What I Learned
- Working with REST APIs
- React Router and dynamic routes
- State management using hooks
- Component-based architecture

## Screens
- Products listing page
- Product detail page
- Cart page

Project is complete and functional.
